# Task: How to filter issues by Assignee in Linear ?

**Application:** Linear

## Steps Executed

1. Go to Issues: Click on "Issues" in the main sidebar on the left.
2. Open Filters: Click on the "Filter" button, typically located above the list of issues.
3. Select Assignee Filter: In the filter menu that appears, click on "Assignee".
4. Choose Assignee: A dropdown will appear where you can select the specific assignee you want to filter by.
