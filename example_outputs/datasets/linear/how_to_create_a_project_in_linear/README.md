# Task: How to create a project in Linear?

**Application:** Linear

## Steps Executed

1. Go to Projects: Click on Projects in the main sidebar on the left.
2. Click New Project: Find and click the New project button, usually located in the top right corner.
3. Name Your Project: Type in the Project name field.
4. Click Create Project: Optionally add a description, team, or dates, then click Create project.
