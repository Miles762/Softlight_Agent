# Task: Create a project in Linear named "hello world"

**Application:** Linear

## Steps Executed

1. Go to Projects: Click on Projects in the sidebar on the left.
2. Click New project: Find and click the 'New project' button, usually in the top right.
3. Name the project: Type "hello world" into the Project name field.
4. Click Create project: Optionally add a description or team, then click the 'Create project' button.
